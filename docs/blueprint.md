# **App Name**: GDP Insights

## Core Features:

- Data Table Display: Display GDP records in a sortable, paginated data table.
- Add New Record Form: Provide a user-friendly form to add new GDP records with validation.
- Create GDP Record: Implement a function to create new GDP records in Firestore.
- Read GDP Records: Implement real-time fetching of GDP records from Firestore for display.
- Update GDP Record: Enable editing of GDP values directly in the data table.
- Delete GDP Record: Implement a secure delete function with confirmation for removing records.
- Trends Analysis Tool: Use gen AI tool to analyze GDP data to identify growth trends and potential economic insights, using the year and value properties.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5) to convey professionalism and stability.
- Background color: Very light gray (#F5F5F5), nearly white, for a clean and modern look.
- Accent color: A shade of purple (#7E57C2), analogous to the primary, to highlight interactive elements.
- Font pairing: 'Inter' (sans-serif) for body and headlines
- Use minimalist icons for actions like edit and delete.
- Maintain a clean, dashboard-style layout with clear sections for data display and form input.
- Subtle animations on data updates and user interactions to enhance the user experience.
